Build a production-ready reusable ecommerce platform using Next.js for small and sole-owner businesses.

The platform must be designed as a generic single-tenant ecommerce solution that can be deployed independently for different businesses using the same codebase with customizable settings and branding.

TECH STACK

Frontend:
- Next.js latest App Router
- TypeScript
- TailwindCSS
- Zustand or Redux Toolkit for state management
- TanStack Query for server state management
- React Hook Form + Zod for forms and validation

Backend:
- Next.js API routes or modular backend architecture inside Next.js
- mongoose ORM
- MongoDB database

Authentication:
- JWT authentication with refresh token support
- Role-based authentication
- Protected admin routes
- Password hashing using bcrypt

Storage:
- Cloudinary integration for image uploads

Payments:
- Monnify payment gateway integration
- Payment initialization
- Payment verification
- Webhook verification
- Transaction re-query
- Failed payment handling
- Duplicate webhook protection

GENERAL REQUIREMENTS

- Mobile-first responsive design
- Production-ready folder structure
- Reusable architecture
- Modular component structure
- Clean scalable architecture
- SEO-friendly
- Optimized performance
- Image optimization
